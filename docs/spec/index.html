3계층 의미 누층

캡처는 즉답으로 끝나지 않고, 자정마다 분류·다이제스트로 의미가 쌓입니다.

Layer 1 · 인입 (실시간)

캡처 → VLM 캡션 + LLM 즉답 → Record. 베르(인사이트) + 쿠키(빠른 한마디).

Layer 2 · 분류 (자정)

Record에 thread·type·tags·위치 부착. 며칠 무언급 thread 점검.

Layer 3 · 다이제스트 (자정)

일/주/월 서술 저널 + 상위 인덱스 생성. 조간·석간 발행물.
