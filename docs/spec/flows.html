유저플로우

단일 경로는 번호 스텝, 분기·상태 전이가 있는 흐름은 다이어그램으로 (NNG 가이드 적용). 화면은 화면 인벤토리.

플로우 인덱스
  • A · 캡처 → 즉답
  • B · 알림 → 기록
  • C · 차량 출차/주차
  • D · 대화 + 액션
  • E · 검색/회상
  • F · 신호 → 데스크
  • G · 도착 말걸기

A · 캡처 → 즉답

flowchart TD s([기록 탭 · 카메라]) --> cap[사진/영상/음성/텍스트 입력] cap --> send[전송 → POST /ingest async] send --> stub[stub record processing 즉시 표시] stub --> poll{4s 폴링
status=done?} poll -- 아니오 --> poll poll -- 예 --> show[베르 인사이트 + 쿠키 한마디 + 분석] show --> react[흐름서 반응·편집 가능] react --> e([끝])

B · 알림 → 기록 (동반자 질문에 답)

  1. 수집기가 이벤트 감지(도착·체크인·출차 등) → 동반자가 선제 멘트 생성
  2. 폰에 알림 표시 — payload {s:화자, t:질문}
  3. 알림 탭(콜드스타트/포그라운드) → 앱이 payload 수신
  4. 기존 모달 정리 후 기록 탭으로 이동, 상단에 질문 배너 표시
  5. 아빠가 사진/음성/텍스트로 답 입력
  6. 전송 → /ingest에 companion_prompt/speaker 동봉(맥락 포함)
  7. 흐름에 질문 버블과 답 record가 쌍으로 남는다(탭한 시각 기준)

C · 차량 출차/주차 (2상태 머신)

물리 전이는 수집기, "어디 가에 답했나"는 백엔드 운행 스레드가 판정. 상세는 Tech Spec.

stateDiagram-v2 [*] --> 주차중 주차중 --> 운전중: 차 BT 연결 + 200m↑ 이동
→ "어디 가?" 운전중 --> 주차중: BT 해제(2틱)
→ 주차 GPS + "잘 도착했어?/어디?" 운전중 --> 주차중: 30분 정지(안전망)
→ 조용히 리셋
"잘 도착했어?" 분기 주차 시 백엔드가 question_ts 이후 첫 user 답을 찾는다 → 있으면 "(목적지) 잘 도착했어?", 없으면 "어디 세웠어?".

D · 대화 + 액션 확인

flowchart TD i[흐름 입력] --> send[POST /chat] send --> resp[베르 응답 + 근거 record] resp --> act{액션 제안
있나?} act -- 없음 --> done([끝]) act -- 있음 --> ask[버튼: 일정 등록할까요?] ask --> dec{확인?} dec -- 확인 --> run[confirm → 캘린더 생성] dec -- 취소 --> cancel[cancel → 무효] run --> done cancel --> done

E · 검색/회상

  1. AppBar 검색 → 질의 화면
  2. "지난주 마우스 관련?" 입력 → POST /query
  3. 3요소 검색으로 후보 record/journal 수집 → LLM이 답 합성
  4. 답변 + 참조 record 썸네일 카드 표시(누적 히스토리)
  5. 다음 질문 반복

F · 신호 → 데스크 처리

  1. 수집기 30분 sync → /signals/sync(SMS·부재중·알림)
  2. 새 신호만 로컬 LLM이 5단계 분류 → brief 생성 → 폰 알림
  3. 데스크 탭: 당장 처리(action)·리마인더·오늘 알림·가계부·오래 못 챙긴 사람
  4. action 카드 확인 → dismiss(실행취소 가능)
  5. 분류 틀리면 재지정(recategorize) / '부정확' 피드백
  6. 결제는 가계부로, 처리할 일은 리마인더로 자동/수동 승격

G · 도착 말걸기 (banter)

  1. 수집기 체류·도착 감지(BT→WiFi→GPS)
  2. 거주자 장소(집=쿠키·작업실=베르)면 그 주인이 먼저 반김
  3. 게이팅(마스터·새벽 침묵·쿨다운) 통과 시에만 알림
  4. 베르·쿠키가 흐름에서 2~3턴 도란도란(사용자 미포함) → 흐름 캡션 "( ~에 도착했다 )"
  5. 미등록 새 곳 15분+ 체류 → "여기 어디?" 질문 → 답하면 임시 장소 저장
흐름의 권한·게이팅 정책은 Tech Spec, 화면 단위는 디자이너 문서.
