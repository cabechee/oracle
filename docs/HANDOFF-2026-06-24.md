### B. 범용 재처리(코멘트 반영) — 핵심 요청

- 설계: 각 생성기에 `comment=""` 파라미터 → 프롬프트에 `[사용자 피드백 — 반영해 다시 써라]\n{comment}` 주입. 기존 엔드포인트(`/digest/run`·`/briefing/run`·`/discovery/run`·`/records/{id}/reprocess`)에 comment 추가 + 어드민 UI.

  - **확인 필요**: "어제로부터"가 정확히 뭔지(조간의 어제 요약? 별도 화면?) 사용자에게.
